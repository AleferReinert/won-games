## Boilerplate

Utilizando TypeScript, React, NextJS e Styled Components.

## Iniciando
`yarn create next-app -e https://github.com/AleferReinert/boilerplate`

Obs.: o repositório precisa estar público.

## Storybook
Após rodar `yarn build-storybook`, executar: 
- `cd storybook-static`
- `http-server`
